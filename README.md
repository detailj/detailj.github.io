# amazeUI 博客/资讯类前端模板

amazeUI

---

这是一个简单但又全面的前端博客/资讯类模板

**[演示点我][1]**

## 简单的介绍：

- 响应式布局，3个媒体查询断点
- 多种布局方案，主页四种布局，易扩展，易开发更加复杂页面。文章页3种布局，包含最常用的布局方案。
- 组件化
- 基于amazeUI的优秀特性上，增强用户体验和美观性
